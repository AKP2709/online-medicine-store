# Online Medicine Store üè•

A Spring Boot project for managing online medicine sales and orders.

## Features
- Admin: Add, update, delete medicines
- Customer: Register, login, place orders, view order history
- Tech: Java 17, Spring Boot, Spring Data JPA, MySQL

## How to Run
1. Configure MySQL in `application.properties`
2. Run `MedstoreApplication.java`
3. Use Postman or Swagger to test APIs

## API Endpoints
| Method | URL | Description |
|--------|-----|-------------|
| POST   | `/admin/medicines` | Add new medicine |
